Reclass process for future evaluation propose.

A man made evaluation index is made for the future evaluation, the reclass function in the Spatial Analyst Tools toolbox - Reclass toolset is used to assign those man made evaluation index.

In my case, the evaluation index are started from 0 to 10. The higher values will be given to attributes that are more suitable for building a school.

Following evaluation criteria is used:

  • The slope should be in a low level
  • It should close to the recreation site
  • It should far away from the current school

The model builder is arranged into the following fashion:

All three input raster are reclassed using the *Equal Interval* method, and broke down the 10 classes. For the slope and recreation site process, the new values assigned from reclassification are reversed, since the smaller value means a more suitable environment of school building.

Weighted and combing the current dataset

Following percentage of influence was decided on all above three area(school, slope, recreation site).

  • Distance to recreation site: 50%
  • Distance to school: 25%
  • Slope: 13%
  • Landuse: 12%

The original evaluation scales are changed from "1 to 9 by 1" to "1 to 10 by 1".

Due to limitation on construction, certain slope could not to use for building a school, a restriction was made on Reclassed slope that has evaluation value from 1 to 3.

Similar process was used for land use, too. Water, Wetlands are restricted during the evaluation. And the evaluation index for landuse are assigned by following:

  • Brush/transitional : 5
  • Barren land: 10
  • Built up: 3
  • Agriculture: 9
  • Forest: 4
